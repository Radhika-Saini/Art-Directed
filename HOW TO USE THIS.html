Let me know if you like it, or have any helpful suggestions. travisnelison@gmail.com HOW TO USE THIS: 1 Place this folder in your wp-content/themes/ folder, and turn the theme on in the Wordpress backend. 2 Place Art-Directed post files in the /custom-posts/ folder. See an example or to already there. 3 You need to link up the post files to the WordPress database for each post. To do that, start a new post and create a Custom Field named "custom-file" the value of that field needs to be the location of the post files relative to the theme folder. (e.g. "001-clown-fish/clown-fish.php") 4 Now you can use your own code editor to create a custom post and share your awesomeness with the universe. A FEW TIPS: - In the post files (clown-fish.php) you should refer to the posts assets by using the template_url WordPress code to make your post relative to the theme folder. (Example: ) - Just to emphasize the above, it's a good idea to keep ALL the assets that the post needs in it's post folder. Put all the css, js, images, fonts and so forth in that folder. It will help in the long run. - Just to make sure this is clear, this theme isn't about any specific style. It's about the process I use to do Art-Directed posts. You need to use your own styles.
